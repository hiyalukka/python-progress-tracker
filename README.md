#  Python Progress Tracker

A simple command-line tool built using Python fundamentals to track daily study sessions and stay consistent with learning goals.

---

##  Features

* Add daily study sessions (topic + hours + date)
* Store progress in a file (`progress.txt`)
* View all past study logs
* Calculate total hours studied

---

##  Tech Used

* Python (Basics)

  * Variables
  * Loops
  * Conditionals
  * File Handling
